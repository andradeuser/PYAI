# PYAI

Projeto desenvolvido para a disciplina de **Versionamento de Código** da Especialização em Inteligência Artificial Generativa (GENAI) da Universidade Federal do Paraná (UFPR).

## Sobre o Projeto

Este repositório contém os artefatos e exercícios práticos relacionados à disciplina de Versionamento de Código, abordando conceitos e ferramentas essenciais para o controle de versão em projetos de software.

## Tecnologias Utilizadas

- Python
- Git

## Pré-requisitos

- [Python 3.8+](https://www.python.org/downloads/)
- [Git](https://git-scm.com/)

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/PYAI.git
   ```
2. Acesse o diretório do projeto:
   ```bash
   cd PYAI
   ```
